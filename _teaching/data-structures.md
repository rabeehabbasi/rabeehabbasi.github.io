Aims and Objectives
======
Data structures are used to store and retrieve data efficiently. The main purpose of this course is to understand basic data structures and algorithms associated with these data structures. The course covers a wide range of data structures and their applications for solving various problems. At the end of the course, students should be able to implement basic data structures, compare performance of various data structures, and use appropriate data structures to solve a variety of problems.

Lectures
======
1. Introduction to the course
1. Time complexity: Linear Search, Comparing Time Complexities
1. Time complexity: Asymptotic time complexity notations and analysis; rate of growth; logrithmic time complexities
1. Sorting algorithms selection, insertion, and bubble sort and their time complexities.
1. Stack ADT implementation using arrays.
1. Stack ADT implementation: comparison operators: operator==, operator<
1. Stack ADT implementation using linked structures: constructor, push, top, size, empty.
1. Implementation of linked-structures-based stack ADT: pop, destructor, operator=, and copy constructor 
1. Applications of Stack: Infix to Postfix conversion and Postfix evaluation
1. Queue ADT implementation using arrays (linked-based covered in lab)
1. Singly linked list (forward_list in STL). Implementation using a circular linked list with dummy head node.
1. Implementing STL like iterators in singly linked list. Nested classes. Overloading operator*, operator++, and operator++(int) for forward_list
1. Forward_list: implementing operator->, operator==, operator!=. Functions using iterators in forward_list: insert_after, erase_after.
1. Midterm Exam
1. Doubly Linked List/ std::list in STL: Structure of a doubly linked list in STL; Doubly circular linked list with a dummy head node; push_front; pop_front; push_back; pop_back;
1. Implementing iterators (forward / reverse) for the doubly linked list; implementing insert function for inserting a value before a node; 
1. Binary Search Tree: graph, tree, binary tree, binary search tree, full tree, minimum height, maximum height, inserting a value in a tree
1. Introduction to Binary Search Trees in STL (set, multiset, map, multimap); Implementation of std::set in STL; implementing insert function for set
1. Deleting a node in a BST. Implementing erase function
1. Traversals in BST: in-order iterative traversal)
1. Traversals in BST: Recursive traversals (in-order, pre-order, post-order)
1. Balanced BST: AVL insertion and deletion
1. Implementing key/value pairs in BST (map); Implementing at and operator[] functions for map ADT
1. Heap: properties, complete binary tree, ReheapUp, ReheapDown
1. Heap Sort. Priority Queue.
1. Hashing: direct addressing, collisions, chaining, search, delete; properties of good hash functions;  Chaining-based hashing
1. Hashing: How hashing (unordered_map) is implemented in STL? 
1. Hashing: Implementing STL like iterators
1. Hashing: hashing strings, open addressing: linear probing, quadratic probing, double hashing;
1. Graphs: introduction, applications, time complexity of basic graph functions, graph representation (adjacency list, adjacency matrix)
1. Graphs: adjacency list based implementation; Breadth First Search (BFS); Depth First Search;
